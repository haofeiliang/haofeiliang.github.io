Reference implementation of the **InstantOMR** oblivious message retrieval scheme. Supports low latency and high parallelism, built on the TFHE fully homomorphic encryption scheme.

- Lead developer and first author of the corresponding USENIX Security '26 paper
- Implements efficient functional bootstrapping-based message retrieval
